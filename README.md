# 長野CAKE  TEAM　ZONe

## 説明

### 長野県にある洋菓子屋「長野CAKE」のECサイト
 ・通販のみの完全受注型生産
 
 ・送料は全国一律８００円
 
 ・支払いはクレジットカードか現金振り込みの２つから選択ができる
 
 ・家族や友人へのプレゼントなど、注文者の住所以外にも直接送ることが可能
